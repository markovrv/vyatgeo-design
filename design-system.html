Анимации

.anim-fade-up — появление снизу (opacity + translateY). Подключается через IntersectionObserver.

.anim-fade-up-delay-1 ... delay-6 — задержка 0.1–0.6 с для каскадных анимаций.

.anim-fade-up { opacity: 0; transform: translateY(30px); transition: 0.6s ease; }
.anim-fade-up.visible { opacity: 1; transform: translateY(0); }
